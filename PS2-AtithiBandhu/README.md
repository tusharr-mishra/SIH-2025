# AtithiBandhu — Smart Tourist Safety System

**Smart India Hackathon 2025 · Problem Statement SIH25002**
**Theme: Travel & Tourism · Team: Quantum Ledgers**

A blockchain-powered, privacy-first tourist safety system using
AI-driven geofencing, verifiable digital IDs and real-time incident response.

---

## The Problem

India receives millions of tourists annually with no unified,
secure system for identity verification or real-time safety monitoring.
Existing systems rely on centralised databases with privacy risks
and have no offline fallback in low-connectivity tourist areas.

---

## Our Solution — AtithiBandhu
Tourist Arrives
↓
W3C Verifiable Digital ID Issued (KYC-based)
↓
Identity Anchored on Hyperledger Fabric Blockchain
↓
AI Geofencing + Anomaly Detection (active monitoring)
↓
SOS Triggered → ERSS-112 + Multi-Agency Dashboard
↓
Session Ends → Credentials Revoked on Departure

---

## Key Features

| Feature | How |
|--------|-----|
| Digital Tourist ID | W3C Verifiable Credentials, session-based |
| Blockchain backbone | Hyperledger Fabric for tamper-proof logs |
| Privacy by design | BBS+ Zero-Knowledge Proofs, DPDP compliant |
| Offline verification | Works without internet in remote areas |
| Emergency response | ERSS-112 integration with fast dispatch |
| Identity options | Aadhaar e-KYC (domestic) + Passport (international) |
